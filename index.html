About Me

Hello, I'm Sam, a student who is passionate about programming and tech. Here's what I did. I started my programming journey in at the age of 10, when my teacher introduced me to Hour of Code. I was fascinated by both the simplicity and complexity of code. Scratch was my first big platform for programming, I would explore community projects, learn for them and create my own spinoffs. Scratch was a great platform for me to learn the basics of code.

As code got more and more complex, Scratch was too limiting to me. I switched to text-based coding and as a beginner, I started with Python. I created some console programs, mostly simple calculators and storytelling, then discovered Discord; Its bot API sparked my interest, it was an in-world use for code, which I could share. Discord bot development allowed me to further enhance my knowledge, and it allowed me to understand common and best practices when writing code.

I wanted to explore something else. I discovered web development, it was something very useful. At first, it was really hard, but with the help of some libiaries such as Tailwind CSS and lots of tutorials, I successfully made some simple static sites. I then had to learn JavaScript as I wanted to explore deeper into web development. After some struggles, I was able to use simple APIs like Open-Meteo to integrate weather data into my site. I then learned backend web development. I used Python, and although it's not the best language for this use case, it was the most comfortable language for me. I used Flask to create dynamic sites, and I learned how to use databases such as MongoDB.

Fast forward to now, here I am, working on my own personal website. My own personal website allows me to share what I do and what I'm passionate about. I am always learning new things and exploring. Recently, I've been interested in artificial intelligence, I explored the NEAT algorithim. I've also been learning Java and expermenting in making mods for Minecraft. A Private Discord Utility Bot
